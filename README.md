# GuruVerse

Platform pembelajaran online untuk guru dan siswa.

## Setup

```bash
git clone https://github.com/kylanazwa0-lab/guruverse.git
cd guruverse
composer install
npm install
```

## Struktur Folder

- `admin/` - Admin dashboard
- `api/` - API endpoints
- `backend/` - Laravel backend
- `database/` - Database scripts
- `guru-belajar/` - Learning platform
- `modules/` - Platform modules

## Dokumentasi

Lihat folder `backend/` untuk dokumentasi setup lebih lanjut.
